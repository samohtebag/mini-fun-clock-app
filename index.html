When creating passwords, always make sure you use something like a password generator. This will help you create a strong password.

Setup 2FA/MFA for stronger authentication.

Setting up a second layer of authentication for logins is an excellent way to protect your assets. In general, most 2FA uses SMS or checking a secondary device for login confirmation.

Consider using an authenticator app

Apps like Google Authenticator or LassPass Authenticator allow you to generate temporary login codes, if the web application allows this, to login. This is one of the most secure ways to protect your assets.
